Equivalente a seguinte operação em C:



resultado = (A0 + (_a * _b)) * 4 - 2) / 2;

if( resultado >= 0 )

  printf("%d\n1\n", resultado);
  
else

  pprintf("%d\n-1\n", resultado);



Onde A0 é o registrador A0, _a e _b são valores guardados em memoria pelo .data.
